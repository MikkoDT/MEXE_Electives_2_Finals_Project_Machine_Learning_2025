# ▶️ How to Run the Project (VS Code + Jupyter)

This guide explains how to run the project using **Visual Studio Code** and **Jupyter Notebook**.

---

##  Software Requirements
- **Python 3.13**
- **Visual Studio Code (VS Code)**
- **Python Extension for VS Code** (by Microsoft)
- **Jupyter Extension for VS Code**

---

## Open the Project

1. Open **Visual Studio Code**
2. Click **File → Open Folder**
3. Select the project folder
4. Open the terminal using **Ctrl + `**

---

## Install Required Python Libraries

Run the following command in the VS Code terminal (PowerShell or CMD):

python -m pip install pandas numpy matplotlib scikit-learn ipykernel
