# Project Requirements

## Software
- Python 3.13
- Visual Studio Code (VS Code)

## VS Code Extensions
- Python Extension (Microsoft)
- Jupyter Extension (for `.ipynb` support)

## Python Libraries
- pandas
- numpy
- matplotlib
- scikit-learn
- ipykernel

## Environment Notes
- The Python interpreter and Jupyter kernel must use the same Python version
- Compatible with Windows, Visual Studio Code, and Google Colab
